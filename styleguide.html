07 / motion

page entrances use fade + a slight vertical shift (0.5s). sections below the fold reveal as they enter the viewport via intersectionobserver. everything respects prefers-reduced-motion.

/* entrance animation */
@keyframes fade-up {
  from { opacity: 0; transform: translateY(10px); }
  to   { opacity: 1; transform: translateY(0); }
}

08 / icons

lucide icons inlined as raw svg — no icon font, no extra network request. sized via the .icon class and colored with currentColor, so they always match the surrounding text.

hover to see .hover-lift and .hover-scale in action
